수업 활동

정보보호학전공 2학년으로서 다양한 수업과 과제를 통해 웹 보안 및 사이버 보안 기술을 학습하고 있습니다. 아래는 주요 수업 활동과 프로젝트입니다.

주요 수업 목록

수업명 학기 주요 내용
웹프로그래밍 2025 상반기 HTML, CSS, JS 등을 활용한 웹 페이지 제작
컴퓨터네트워크 2025 상반기 컴퓨터 네트워크 기본 동작 원리와 OSI 7계층에서의 각 계층 별 공격 기법 대응방안 학습
객체지향프로그래밍 2025 상반기 자바 클래스, 상속, 인터페이스 개념을 이해하고 활용
보안자료구조 2025 상반기 데이터를 구조적으로 표현하 방식과 이를 구현하는데 필요한 알고리즘에 대해 학습

주요 과제 및 프로젝트

웹프로그래밍 - 개인 홈페이지 제작

HTML, CSS, JS를 사용해 웹 페이지를 제작하며 웹의 동작방식과 원리에 대해 이해.

보안자료구조 - 스택 자료구조를 활용한 핀볼 게임 구현

스택의 push, pop, peek 연산을 직접 구현하고 스택 자료구조에 대한 이해를 높힘.

컴퓨터네트워크

가상 머신을 활용한 스니핑, 스푸핑 실습



수업 활동의 의의

수업에서 배운 내용들을 이해하고 주어진 과제를 수행하며 해당 지식에 대한 이해는 물론 실제 실습을 통해 실무적인 능력을 향상시킬 수 있었습니다.



사용한 도구

  • VS Code: 코드 제작 및 편집
  • Wireshark: 네트워크 패킷 분석
  • TCPDump: 스니핑 실습
